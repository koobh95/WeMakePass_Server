# WeMakePass
WeMakePass는 자격증 혹은 자격증 시험을 주제로 의견을 나눌 수 있는 커뮤니티인 동시에 모의시험에 도전할 수 있는 어플리케이션입니다. 모의시험에 도전해 내 실력을 확인하고 모르는 문제나 어려웠던 문제가 있었다면 자체 커뮤니티를 통해 같은 목적으로 모인 사람들과 의견을 나눌 수 있습니다.
